# FillWork Project Screen Design Spec

## 목표

Dashboard v1.0의 문서 중심 구조와 현장 모드를 이어받아 프로젝트의 계획, 실행, 현장 증빙을 하나의 작업 공간에서 관리한다.

## 기본 화면 구조

- 좌측 195px: Dashboard와 동일한 글로벌 사이드바
- 상단 53px: 모드 토글, 검색, 알림, 프로필
- 중앙: 프로젝트 헤더와 선택된 업무 뷰
- 우측 340~438px: 업무 상세 또는 현장 컨텍스트 패널

## 프로젝트 헤더

- 프로젝트 아이콘과 프로젝트명
- 상태: 진행 중 / 보류 / 완료
- 진행률과 마감일
- 참여자 아바타
- `새 업무` 기본 버튼
- 보기 전환: 보드 / 리스트 / 캘린더 / 타임라인 / 파일

## 현장 모드

- 상단 모드 토글의 `현장 모드` 선택 상태 유지
- 카드에 방문 위치, 사진 수, 마지막 동기화 시각 표시
- 모바일에서는 업무 카드 하단에 `사진 촬영`, `방문 완료` 주요 액션 배치
- 미동기화 사진과 긴급 업무를 상단 필터로 제공

## 보드 뷰

- 컬럼: 대기 / 진행 중 / 검토 / 완료
- 컬럼 폭: 280px, 간격 12px
- 업무 카드 높이: 최소 132px
- 카드 정보: 제목, 담당자, 마감일, 우선순위, 체크리스트, 첨부 사진 수, 현장 태그
- 검토 컬럼에는 승인 대기 배지 강조

## 리스트 뷰

- 단순 관리 표가 아닌 그룹형 업무 리스트
- 섹션 헤더에 상태, 업무 수, 진행률 표시
- 행 높이 52px
- 제목 왼쪽 체크박스, 오른쪽 담당자·마감일·현장·승인 상태 배치
- 행 확장 시 체크리스트와 최신 사진 미리보기 표시

## 우측 상세 패널

- 폭 380px 권장
- 탭: 개요 / 댓글 / 파일 / 활동
- 업무 설명은 블록형 문서 편집 구조
- 현장 증빙 사진은 3열 썸네일
- AI 영역: 진행 요약, 지연 위험, 다음 행동 추천
- 하단 고정 액션: 완료 처리 / 검토 요청

## AI 프로젝트 코파일럿

- 지연 가능 업무 알림
- 오늘 우선 처리 업무 추천
- 현장 사진 기반 진행 요약
- 주간 프로젝트 보고서 초안
- 검토 요청 전 누락 항목 확인

## Empty 상태

- 프로젝트 없음: 프로젝트 템플릿 선택과 새 프로젝트 생성
- 업무 없음: `첫 업무 만들기`와 AI 업무 목록 생성
- 현장 사진 없음: 업로드 또는 모바일 촬영 안내
- 검색 결과 없음: 필터 초기화 액션

## Skeleton

- 프로젝트 헤더 1개
- 보드 컬럼 4개
- 컬럼당 카드 Skeleton 3개
- 우측 상세 패널 Skeleton
- 실제 카드와 동일한 9px radius 및 낮은 대비 사용

## 반응형

- 1440px 이상: 보드와 우측 상세 패널 동시 표시
- 768~1439px: 우측 패널 Drawer 처리, 사이드바 축소
- 390~767px: 단일 상태 컬럼, 하단 보기 전환, 현장 액션 고정
