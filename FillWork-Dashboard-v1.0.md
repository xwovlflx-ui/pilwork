# FillWork Dashboard v1.0

## 기준

- 화면: Desktop Dashboard
- 뷰포트: 1536 x 1024
- 상태: UI 및 Mock Data 전용
- 디자인 비율: Notion 50% / Toss 30% / Slack 20%
- 최종 시안: `output/playwright/FillWork-Dashboard-v1.0.png`

## 핵심 정체성

FillWork Dashboard는 문서 작성, 현장 업무, 사진 업로드, AI 지원을 한 화면에서 연결한다. 일반적인 관리자 통계보다 사용자가 오늘 처리할 문서와 현장 흐름을 우선 노출한다.

## v1.0 구성

1. 문서 중심 최근 페이지
2. 사무실 모드 / 현장 모드 전환
3. 적극형 AI 업무비서
4. 오늘의 현장 상태 카드
5. 현장 업로드 센터
6. 긴급 공지와 방문 일정
7. 캘린더와 즐겨찾기

## 적극형 AI 비서

- 미처리 업무 경고
- 다음 업무 추천
- 공지 초안 생성 제안
- 주간 생산성 분석

## 현장 업로드 센터

- 오늘 업로드 사진 수
- 현장별 자동 분류
- AI 요약 처리 상태
- 관리자 승인 대기 현황
- 전체 처리율 진행 표시
