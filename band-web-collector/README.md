# BAND Web Collector

웹 브라우저에서 사용자가 직접 로그인한 BAND 웹 화면을 Selenium으로 조작해, 본인이 접근 권한을 가진 밴드 1개의 게시글 내용을 엑셀로 정리하는 프로그램입니다.

## 수집 대상
- 밴드 1개
- 게시글 화면 텍스트
- 작성자/날짜/본문은 화면 구조에 따라 자동 추출 또는 원문 텍스트로 저장
- 사진/첨부파일은 1차 버전에서 직접 다운로드 대신 스크린샷 경로 저장

## 원칙
- 비밀번호 저장 안 함
- 로그인은 사용자가 직접 진행
- 권한 없는 밴드 접근/우회 금지
- 본인이 볼 수 있는 자료 정리용

## 설치
```bash
pip install -r requirements.txt
```

Chrome 브라우저가 설치되어 있어야 합니다.

## 실행
```bash
python main.py
```

실행 후:
1. Chrome 창이 열립니다.
2. BAND에 직접 로그인합니다.
3. 수집할 밴드방으로 직접 이동합니다.
4. 터미널에서 Enter를 누릅니다.
5. 프로그램이 스크롤하며 화면 텍스트와 스크린샷을 저장합니다.
