# FillWork Project Kanban v1

## 화면 기준

- 경로: `/projects`
- Desktop: 1536 x 1024
- 데이터: 로컬 Mock Data 8건
- 시안: `output/playwright/FillWork-Project-Kanban-v1.png`

## 칸반 구조

| 컬럼 | 초기 업무 수 | 목적 |
| --- | ---: | --- |
| 할 일 | 3 | 시작 전 업무 관리 |
| 진행 중 | 3 | 현재 수행 업무 관리 |
| 완료 | 2 | 마무리 업무 기록 |

## 카드 정보

- 업무 제목
- 담당자 이름과 아바타
- 마감일
- 우선순위: 긴급 / 높음 / 보통 / 낮음
- 업무 태그
- 현장 위치
- 첨부파일 수

## 인터랙션

- HTML5 Drag & Drop으로 컬럼 간 이동
- 드래그 진입 컬럼 테두리 및 배경 강조
- 이동 즉시 컬럼별 업무 수 갱신
- 새 업무, 필터, 담당자, 보기 전환 UI 제공
- 기능 데이터는 새로고침 시 초기 Mock Data로 복원

## 검증

- `할 일` 첫 카드를 `진행 중`으로 이동
- 할 일: 3개 → 2개
- 진행 중: 3개 → 4개
- 프로덕션 빌드 성공
- 브라우저 콘솔 오류 0건
